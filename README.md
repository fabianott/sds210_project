# ZüriWieNeu

## Beschreibung
Ziel dieses SDS-Projekts ist die Analyse des Datensatzes **Züri wie neu** von Open Data Zürich, welcher sämtliche erfassten Schäden an der städtischen Infrastruktur Zürichs beinhaltet. Dabei wird insbesondere untersucht welche räumliche und zeitliche Muster zu erkennen sind.

## Daten Quellen
- Zürich wie neu: https://data.stadt-zuerich.ch/dataset/geo_zueri_wie_neu
- Geodaten (Quartiere der Stadt Zürich): https://data.stadt-zuerich.ch/dataset/geo_statistische_quartiere
